Projeto de um jogo básico, levemente inspirado em *Vampire Survivors*, onde o jogador precisa sobreviver a ondas de inimigos.  
O objetivo deste projeto é treinar conceitos de **Programação Orientada a Objetos (POO)** aplicados a um jogo 2D e aproveitar para conhecer o ambiente do Github.


Por enquanto, o jogo possui as seguintes funcionalidaes:

- Controle de personagem com teclado (movimento e ataque)
- Inimigos que aparecem aleatoriamente na tela
- Tela de game over e reinício do jogo
- Estrutura em POO (classes para jogador, inimigos, armas, etc.)
